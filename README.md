## ⚙️ Executando <a name="running"></a>

```bash

# Baixe este repositório ou clone pelo Git usando o comando:
$ git clone https://github.com/drisabelles/frontmentor-space-tourism.git

# Acesse a pasta do projeto
$ cd frontmentor-space-tourism

# Instale as dependências
$ npm install

# Inicie o projeto
$ npm start

```
